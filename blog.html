Terminal - Articulo 3

Interfaz de línea de comandos

La terminal en Linux es una interfaz de línea de comandos (CLI) basada en texto que permite
a los usuarios interactuar directamente con el sistema operativo escribiendo comandos para
ejecutar tareas como gestionar archivos, instalar programas, controlar procesos y automatizar
operaciones, siendo una herramienta poderosa para administradores y desarrolladores, aunque
no esencial para el uso diario gráfico.
